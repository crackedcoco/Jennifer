# Weekly Analytics Brief

Full week-over-week analytics comparison. Run on Monday mornings.

## Steps

1. Use `ga4_traffic_report` to pull last 7 days of data.

2. Compare against the prior 7 days.

3. Build a comprehensive report covering:
   - **Total sessions** — this week vs last week (% change)
   - **Unique users** — this week vs last week
   - **Top 10 pages** by traffic with week-over-week change
   - **Traffic sources** — breakdown by channel (organic, direct, referral, social, paid)
   - **Top referrers** — which external sites sent the most traffic
   - **Conversion metrics** — if available

4. **Identify trends:**
   - Pages gaining momentum (biggest % increase)
   - Pages losing traffic (biggest % decrease)
   - New pages that appeared in top 50 this week
   - Channel shifts (e.g., organic up but social down)

5. **Send to Discord** with clear formatting:
```
📊 Weekly Analytics — [date range]

📈 HEADLINE
[1-2 sentence summary of the week]

🔢 KEY METRICS
Sessions: [X] ([+/-]%)
Users: [X] ([+/-]%)
Pageviews: [X] ([+/-]%)

🏆 TOP PAGES
[top 10 with trends]

📡 TRAFFIC SOURCES
[channel breakdown with changes]

💡 INSIGHTS
[2-3 actionable observations]
```
