# Morning Briefing

Run the full morning briefing and send it to me on Discord.

## Steps

1. **Calendar** — Use the `daily_agenda` tool to get today's events. Format as a clean schedule with times and meeting links.

2. **Email Digest** — Use the `email_digest` tool to get categorized unread emails. Show counts per category and list the urgent and actionable ones with sender + subject.

3. **Analytics Snapshot** — Use the `ga4_traffic_report` tool to get yesterday's traffic. Highlight any notable changes vs the prior day.

4. **Send to Discord** — Combine everything into a single well-formatted message. Use this structure:

```
☀️ Morning Briefing — [date]

📅 TODAY'S SCHEDULE
[events listed by time]

📧 EMAIL ([count] unread)
🔴 Urgent: [list]
🟡 Actionable: [list]
📋 FYI: [count] items

📊 YESTERDAY'S TRAFFIC
[key metrics and changes]

🎯 TOP PRIORITIES
[suggest 3 priorities based on calendar + email]
```
