Quick Start

Get Up and Running

Six steps from zero to a fully autonomous agent working in the background.

1

Clone the repo

Pull the Jennifer repository to your local machine.

git clone https://github.com/crackedcoco/Jennifer.git
2

Install dependencies

Install Jennifer and all required packages in development mode.

pip install -e ".[dev]"
3

Set up Google OAuth credentials

Create a Google Cloud project, enable Gmail and Calendar APIs, and download your OAuth credentials JSON.

4

Configure Discord bot

Create a Discord application, add a bot, and set the token in your environment variables.

5

Launch Jennifer

Start Claude Code with the Discord channel plugin to bridge commands to your phone.

claude --channels plugin:discord@claude-plugins-official
6

Start loops from Discord

Use Discord commands to activate automation loops. Jennifer takes it from here.
